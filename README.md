# 随机数生成器

这是一个本地 HTML 工具。直接用浏览器打开 `index.html` 即可使用，不需要安装依赖或启动服务。

详细按钮说明和完整使用流程见 `软件使用说明.md`。

## 主要功能

- 按范围生成随机整数或小数。
- 支持参考数上下平衡生成。
- 支持不允许重复结果。
- 支持随机种子 Seed：同样条件和 Seed 会生成同样结果，便于科研复现。
- 支持随机分组：可把样本编号随机分配到多个组。
- 支持统计摘要：数量、最小值、最大值、均值、中位数、标准差、Q1/Q3。
- 支持参数元数据：生成时间、模式、Seed、关键参数。
- 支持历史记录：最多回看最近 10 次生成结果。
- 支持一键清除历史记录。
- 支持浅色模式 / 深色模式切换。
- 支持中文 / English 界面切换，默认中文。
- 页面主要操作按钮和区域标题加入轻量图标，提升识别度。
- 支持复制全部、复制单项、导出 TXT、导出 CSV。
- CSV 支持三种格式：编号 + 结果、仅结果一列、参数元数据 + 结果。

## 使用建议

### 普通随机数

1. 选择“普通范围”。
2. 设置最小值、最大值、生成数量。
3. 选择整数或小数。
4. 如需可复现，填写 Seed。
5. 点击“生成”。

### 参考数平衡

1. 选择“参考数平衡”。
2. 设置范围、生成数量和参考数。
3. 工具会尽量在参考数上下两侧平衡生成。

### 随机分组

1. 选择“随机分组”。
2. 设置样本数量。
3. 每行输入一个组名。
4. 选择“均衡分配”或“完全随机”。
5. 如需复现分组结果，填写 Seed。

## 科研记录建议

- 正式使用时建议填写 Seed。
- 导出时优先选择“参数元数据 + 结果”，方便以后说明随机结果来源。
- 论文或实验记录中可保存：Seed、范围、数量、模式、生成时间。

## 限制

- Seed 模式使用确定性伪随机算法，适合复现实验随机序列，不用于密码学安全场景。
- 留空 Seed 时使用浏览器内置 `Math.random()`。
- 当前版本未加入正态分布、区组随机化、排除值、步长、历史记录等高级功能。

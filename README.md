# 随机数生成器

这是一个本地 HTML 随机数生成器。直接用浏览器打开 `index.html` 即可使用，不需要联网、安装依赖或启动服务。

详细按钮说明和完整使用流程见 `软件使用说明.md`。

## 主要功能

- 按范围生成随机整数或小数。
- 支持参考数上下平衡生成。
- 支持随机分组，并为每个样本生成一个随机数字。
- 支持随机种子 Seed：同样条件和 Seed 会生成同样结果，便于记录和复现。
- 支持统计摘要：数量、最小值、最大值、均值、中位数、标准差、Q1/Q3。
- 支持参数元数据：生成时间、模式、Seed、关键参数。
- 支持右上角历史记录下拉菜单，最多回看最近 100 次结果。
- 支持一键清除历史记录。
- 支持浅色模式 / 深色模式切换。
- 支持中文 / English 界面切换，默认中文。
- 支持横向/纵向复制数字、复制单项、导出 TXT、导出 CSV。
- 支持可编辑表头和单元格的表格预览、转置、复制表格、导出 Excel 表格。

## 使用建议

### 普通随机数

1. 选择“普通范围”。
2. 设置最小值、最大值、生成数量。
3. 选择整数或小数。
4. 如需复现结果，填写 Seed。
5. 点击“生成”。

### 参考数平衡

1. 选择“参考数平衡”。
2. 设置范围、生成数量和参考数。
3. 工具会尽量在参考数上下两侧平衡生成。

### 随机分组

1. 选择“随机分组”。
2. 设置样本数量、组名和分配方式。
3. 设置随机数字范围、整数/小数、小数位和是否允许重复。
4. 点击“生成”，结果会包含 `sample_id`、`group`、`random_number`。

## 记录与导出建议

- 需要复现时建议填写 Seed。
- 需要留档时建议导出“参数元数据 + 结果”的 CSV。
- 只需要数字时使用横向/纵向复制；需要表头和完整列信息时使用表格预览里的“复制表格”或“导出Excel表格”。

## 限制

- Seed 模式使用确定性伪随机算法，适合复现随机序列，不用于密码学安全场景。
- 留空 Seed 时使用浏览器内置 `Math.random()`。
- 历史记录只保存在当前页面内，刷新页面后会清空。
- 表格预览导出的 Excel 表格为 `.xlsx` 文件。
